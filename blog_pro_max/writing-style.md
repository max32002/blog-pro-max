# Writing Style Guide — blog-pro-max

這份文件定義了所有由 blog-pro-max 生成的文章必須遵守的品牌語氣與格式規範。

---

## 語氣原則

- **直接、精簡**：每句話有一個目的，不堆砌形容詞
- **有觀點**：提供明確立場，避免模糊的中立說法（如「可能」、「或許有幫助」）
- **不說廢話**：開頭直接切入主題，不寫「在這個快速變遷的時代⋯」類型的引言

---

## 格式規範

- 標題階層最多使用 H1、H2（不使用 H3 以下）
- 段落以 2–4 句為單位，不寫超過 5 行的長段落
- 清單項目每條控制在 20 字以內
- 圖片佔位符格式：`![ALT文字](placeholder-圖片描述)`
- 外部連結格式：`[描述性錨文字](完整 URL)`

---

## 禁止使用詞彙

以下詞彙顯得過時或像 AI 生成，禁止出現：

| 禁用詞彙 | 建議替換 |
|---|---|
| 值得注意的是 | 重點是 |
| 不可否認地 | （直接陳述） |
| 在這個日新月異的時代 | （刪除，直接切入） |
| 顯而易見 | （直接說結論） |
| 毋庸置疑 | （直接陳述） |
| 筆者認為 | 我認為 |
| 相信讀者 | （刪除，直接說） |

---

## 標點符號規範

- 中文語境使用全形標點：`，`、`。`、`：`、`「」`
- 英文專有名詞保留原始大小寫（如 GPT、API、SEO）
- 數字與英文單字前後加半形空格：`使用 3 個步驟` ✓ / `使用3個步驟` ✗

---

## SEO 基本規則

- H1 必須包含核心關鍵字，且全文只出現一次
- 前 100 字內自然帶入核心關鍵字
- Meta description 字數控制在 120–155 字（英文）或 60–80 字（中文）
- 每篇文章至少包含 1 個內部連結佔位符：`[相關文章](internal-link)`
